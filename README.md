Overview

This project was developed as part of an assessment task to design and build a responsive landing page for a Business Networking & Referral Platform.
The platform helps professionals connect, attend structured meetings, exchange referrals, and track business growth.

Features

Responsive and modern landing page design
Hero section with call-to-action buttons
How the platform works section
Key platform features
Target audience section
Testimonials carousel
Call-to-action section
Professional footer with contact information
Mobile-friendly layout
